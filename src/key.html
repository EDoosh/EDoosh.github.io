Key

[] - Required values.
() - Optional values.
[... | ...] - Required to enter one of them.
(... | ...) - Optional to enter one of them.
userRepresentable - A value that represents a user. A mention, name, or userID.
roleRepresentable - A value that represents a role. A name or roleID.
channelRepresentable - A value that represents a channel. A mention, name, or channelID.
